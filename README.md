## 📣 简介

Unofficial API of DeepL translator.[DeepL Translator](https://www.deepl.com/translator)

## 🔰 安装

**pip 安装**
```shell
# windows 用户
$ pip install pydeeplator -U

# Linux / Mac 用户
$ pip3 install pydeeplator -U
```

## 📝 使用

Demo 在源码 example 的目录下。

## ⛏ 代码质量
